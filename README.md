# course_motobike

A motorcycle rent app for customer based on Flutter and Firebase. This is course from [BuildWithAngga - Kelas Online
Full-Stack Flutter Firebase Mobile App Developer: Sewa Motor](https://buildwithangga.com/kelas/full-stack-flutter-firebase-mobile-app-developer-sewa-motor?main_leads=salepage)

## How to Run
1. `git clone https://github.com/FadhilPrawira/course_motobike.git`
2. `cd course_motobike`
3. `flutter pub get`
4. `flutterfire configure`
